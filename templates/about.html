About This Project

DownloadTube is a web app that allows users to download videos from YouTube easily. Whether you're looking to save a tutorial, music video, or any other content, this platform simplifies the process.


How It Works

  1. Paste a video URL.
  2. Select the desired format (MP4 for video or MP3 for audio).
  3. Click the "Start download" button.
  4. Your file will be processed and ready for download shortly.

Building Technologies

This project is built using:

  • Backend: Flask, yt-dlp, APScheduler
  • Frontend: HTML5, CSS, Jinja2
